# Footwear Management Portal 👟

A professional, high-performance web application designed for footwear businesses to streamline inventory management, cataloging, and order processing. This portal allows admins and contributors to maintain a detailed database of footwear products with dynamic attributes and cloud-integrated media.

## 🚀 Purpose
This platform serves as a centralized hub for a client's footwear business. It enables:
- **Order Management**: Facilitates the tracking and organization of product entries.
- **Inventory Oversight**: Admins can monitor entries across all contributors and filter by brand, category, or user.
- **Dynamic Cataloging**: Support for multiple colors, sizes, and custom-defined categories/brands.

## ✨ Key Features
- **Modern Dashboard**: A premium, responsive UI with glassmorphism effects and real-time filtering.
- **Cloud-Based Media**: Seamless image uploads powered by **Cloudinary**, ensuring permanent storage regardless of deployment environment.
- **Smart Filters**: Advanced filtering system to sort products by Brand, Category, and Added By (Admin Only).
- **Customization**: Inline creation of new Brands, Categories, and Sub-categories directly from the entry form.
- **Secure Authentication**: JWT-based login and registration system with role-based access (Admin/Contributor).
- **Mobile Optimized**: Fully responsive design for managing inventory on the go.

## 🛠 Tech Stack
- **Frontend**: HTML5, Vanilla CSS3 (Custom Design System), JavaScript (ES6+).
- **Backend**: Node.js, Express.js.
- **Database**: MongoDB Atlas.
- **Cloud Media**: Cloudinary API.
- **Authentication**: JSON Web Tokens (JWT).
